HOW TO USE NEW FEATURES
═══════════════════════════════════════════════════════════════════════════════

TAGS:
  1. Open task details
  2. Click tag icon
  3. Choose color
  4. Enter tag name
  5. Click "Add Tag"
  Result: Tag appears on task, filterable

SUBTASKS:
  1. Open task details
  2. Click "Subtasks" section
  3. Enter subtask title
  4. Click "+" button
  5. Check off as you complete
  Result: See progress percentage update

TIME TRACKING:
  1. Open task
  2. Click "Start" button
  3. Work on task
  4. Click "Stop" button
  5. Time auto-saves
  Result: View total time in task details

ARCHIVE:
  1. Complete a task
  2. Click "Archive" button
  3. Task moves to archive
  4. Click restore to bring back
  Result: Clean task list, no loss of history

DEVICE DETECTION:
  1. Open app on Device A
  2. Create tasks (saved)
  3. Open app on Device B
  4. Tasks not visible (fresh start)
  5. Back on Device A → tasks visible
  Result: Each device has independent task list

MOBILE MENU:
  1. On mobile screen
  2. Tap hamburger menu
  3. Select action
  4. Menu auto-closes
  Result: Easy access to features

KEYBOARD SHORTCUTS:
  1. Press Ctrl+K (or Cmd+K on Mac)
  2. New task opens
  3. Press Ctrl+Shift+F to search
  4. Press Ctrl+/ for help
  Result: Faster workflow
