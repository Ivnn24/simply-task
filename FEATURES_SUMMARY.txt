// SUMMARY OF NEW FEATURES ADDED

// ========================================
// 1. ENHANCED STORE (src/store.ts)
// ========================================

New Types & Interfaces:
- Subtask: For breaking tasks into smaller parts
- TaskTag: For organizing tasks with categories
- TimeEntry: For tracking work sessions
- RecurrenceType: daily, weekly, monthly, none

New Task Fields:
- tags: string[] - Tag IDs
- subtasks: Subtask[] - Sub-tasks list
- recurrence: RecurrenceType - Recurring pattern
- recurrenceEnd: string - When recurrence ends
- timeEntries: TimeEntry[] - Time tracking entries
- totalTimeSpent: number - Total time in minutes
- isTemplate: boolean - For task templates
- deviceId: string - Device identifier

New Store Actions:
DEVICE MANAGEMENT:
- initializeDevice() - Unique device per browser

TAG MANAGEMENT:
- addTag(name, color) - Create new tag
- removeTag(tagId) - Delete tag
- addTagToTask(taskId, tagId) - Assign tag to task
- removeTagFromTask(taskId, tagId) - Remove tag from task
- setSelectedTags(tagIds) - Filter by tags

SUBTASK MANAGEMENT:
- addSubtask(taskId, title) - Add sub-task
- toggleSubtask(taskId, subtaskId) - Mark complete
- deleteSubtask(taskId, subtaskId) - Remove sub-task

TIME TRACKING:
- startTimer(taskId) - Start tracking time
- stopTimer(taskId) - Stop tracking (auto-saves)
- addTimeEntry(taskId, duration) - Manual time entry

RECURRENCE:
- createRecurringTask(taskId, type, endDate) - Make recurring

ARCHIVE:
- archiveTask(taskId) - Archive completed task
- restoreTask(taskId) - Restore archived task

SEARCH:
- setSearchQuery(query) - Real-time search

DEVICE SYNC:
- Tasks stored by deviceId
- Same device = remembers tasks
- Different device = fresh start
- All local storage (no cloud needed)

// ========================================
// 6. DEVICE DETECTION & LOCAL STORAGE
// ========================================

How it works:
1. On app load, unique deviceId is created
2. Each device gets its own deviceId stored in localStorage
3. Tasks are filtered by deviceId
4. Same browser/device shows saved tasks
5. Different device shows empty (fresh start)
6. No cloud sync - all local

Benefit:
- Privacy: No server storage
- Speed: Instant loading
- Flexibility: Different lists per device
